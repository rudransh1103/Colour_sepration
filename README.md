# Leaf Analysis Tool

Advanced leaf image analysis tool for BTech project.

## Features
- Color separation with RGB replacement
- Boundary and vein detection
- Advanced palette editor
- Auto-save system

## Download
Go to [Releases](../../releases) and download LeafAnalysisTool.exe

## Usage
1. Download the .exe file
2. Double-click to run
3. Upload leaf image and start analysis

No installation or Python required!
